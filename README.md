# Träwelling Android

[![Build Status](https://img.shields.io/badge/status-active-success.svg)](https://github.com/traewelling/android)
[![Kotlin](https://img.shields.io/badge/kotlin-1.9.0-blue.svg)](https://kotlinlang.org)

Eine moderne Android-App für **Träwelling**, die deine Zugreisen smarter und schöner macht. Synchronisiere deine Fahrten, checke bei Freunden ein und behalte den Überblick über deine Timeline – jetzt mit verbesserter Unterstützung für manuelle Zeitanpassungen und detaillierten Fahrplänen.

## 🚀 Features

-   **Echtzeit-Timeline:** Verfolge deine Fahrt mit einer dynamischen Timeline, inklusive Verspätungsanzeige und Bahnsteig-Infos.
-   **Manuelle Zeit-Overrides:** Volle Synchronisation mit der Träwelling-Webseite. Manuell geänderte Abfahrts- und Ankunftszeiten werden priorisiert angezeigt.
-   **Intelligente Haltestellen-Anzeige:** 
    -   Automatische Markierung von **Starthaltestelle** und **Endstation** (auch bei Ausfällen).
    -   Persönliche Badges für **Dein Einstieg** und **Dein Ziel** in hochwertigem Design.
    -   Klare Visualisierung von entfallenen Halten (rot durchgestrichen, prominente Warnung).
-   **Modernes UI:** Gebaut mit Jetpack Compose und Material 3 für ein flüssiges, natives Erlebnis.
-   **Live-Status:** "Live"-Indikator für aktive Fahrten direkt in der Detailansicht.

## 🛠 Installation

1.  Klone das Repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/traewelling-android.git
    ```
2.  Öffne das Projekt in **Android Studio**.
3.  Erstelle eine `local.properties` Datei und füge deinen Träwelling API-Key hinzu (falls erforderlich).
4.  Baue das Projekt mit `./gradlew assembleDebug`.
